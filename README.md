# 💪 FitZone - Fitness Mobile App UI

## 📱 Overview
FitZone is a modern fitness mobile application UI designed to help users achieve their health and fitness goals. The app provides a clean, intuitive, and user-friendly experience for tracking workouts, monitoring progress, and managing fitness activities.

This project was designed in **Figma** as part of my UI/UX design portfolio.

---

## ✨ Features

- Splash Screen
- Welcome Screen
- Login & Sign Up
- Home Dashboard
- Workout Categories
- Workout Details
- Progress Tracking
- Profile
- Settings
- Clean & Modern Interface
- User-Centered Design

---

## 🎨 Design Tools

- Figma
- Adobe Photoshop
- Adobe Illustrator

---

## 🎯 Design Goals

- Simple and intuitive navigation
- Modern and minimal visual style
- Consistent color palette
- Easy-to-use interface
- Better user experience for fitness enthusiasts
